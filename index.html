Frameworks



B

We use Twitter Bootstrap for easy deployment of templated page content in a responsive layout. Please review the documentation thoroughly to familiarize yourself with the affordances of Bootstrap.


We use Google Webfonts for typography. See the Fonts & Typography section below for a specific outline of the use cases for each of our font styles & weights. Our primary font is called Rubik.


We use Font Awesome for CSS-accessible iconography. Much like Bootstrap and Google Webfonts, glyphs from Font Awesome can be hosted remotely and styled via HTML & CSS.
